Dashboard Médecin

Développer le tableau de bord pour les médecins avec une vue d’ensemble des patients. Ce tableau de bord doit afficher les informations essentielles de manière claire et synthétique afin d’améliorer la réactivité et la prise de décision médicale.

Échéance: 08 Juillet 2025

Frontend Dashboard

Exigences

  • Affichage de la liste des patients suivis par le médecin connecté.
  • Résumés des données de chaque patient.
  • Section alertes/notifications médicales.
  • Statistiques synthétiques.

Critères d’acceptation

  • Connexion sécurisée à l’API backend.
  • Affichage responsive pour ordinateur et tablette.
  • Utilisation de graphiques/charts pour certains états (ex: patients par tranche d’âge).
  • Gestion des erreurs (ex: API non disponible).
